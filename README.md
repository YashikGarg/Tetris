# ☁ Controls/Instruction
 
## Main Window
* ↑/↓ To Move Up/Down

* ↵&nbsp;&nbsp;&nbsp; Enter to Select

* Esc To Move Back

## Game Controls

* ← &nbsp;To Move Piece Left

* ↑ &nbsp; To Rotate Piece

* ↓ &nbsp; To Move Down Piece

* → &nbsp;To Move Piece Right

* ⎵ &nbsp;&nbsp;Space To Drop Piece

* P &nbsp;&nbsp;To Pause Game
